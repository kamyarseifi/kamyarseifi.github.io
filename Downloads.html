Last Sound

Last Sound App, is currently for android mobile devices. It is social media application for the last.fm social music website. Graphical Design of the application is completed via CSS as the design of the Web pages. API requests to the servers are done by custom Java libraries, In the front-end another API is used for accessing data's through this libraries. Programs Data is managed through SQLite.  Click on the image below to see the rest of the photos of the app. 

Languages used: 

- HTML5, CSS3, Javascript (Phone-gap)
- Custom Java API Libraries
- SQLite
